# Meu Portfólio --- Desenvolvido com Vue.js

Este é o código-fonte do meu portfólio pessoal, criado com **Vue.js**
para apresentar meus projetos, habilidades e informações sobre mim.

------------------------------------------------------------------------

## Tecnologias Utilizadas

-   **Vue.js 3** (Composition API / Options API)
-   **Vue Router**
-   **HTML5 & CSS3**
-   **JavaScript**
-   **Font Awesome Icons**
-   **Responsividade Mobile-First**

------------------------------------------------------------------------

## Estrutura do Projeto

    src/
    │── assets/          # Imagens, ícones, etc.
    │── components/      # Componentes reutilizáveis (Header, Hero, Cards...)
    │── data/            # Dados dos projetos
    │── views/           # Páginas (Home, Work)
    │── router/          # Configuração das rotas
    │── App.vue          # Componente raiz
    └── main.js          # Arquivo inicial do Vue

------------------------------------------------------------------------

## Como rodar o projeto localmente

### 1. Clone o repositório

``` bash
git clone https://github.com/hmnenj/portfolio.git
```

### 2. Acesse o diretório

``` bash
cd portfolio
```

### 3. Instale as dependências

``` bash
npm install
```

### 4. Rode o servidor de desenvolvimento

``` bash
npm run dev
```

Depois abra no navegador:

    http://localhost:5173
